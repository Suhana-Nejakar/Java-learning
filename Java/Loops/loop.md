# 🔁 Loop Practice Problems (For & While)

👉 Focus on **logic, conditions, and flow control**\
👉 Avoid **built-in shortcuts**\
👉 Try solving each problem using **both `for` and `while` loops**

------------------------------------------------------------------------

## 📘 Section 1: Basic Loop Understanding (Warm-up)

1.  Print numbers from **1 to 10** using a loop.\
2.  Print numbers from **10 to 1** in reverse order.\
3.  Print **even numbers** between **1 and 50**.\
4.  Print **odd numbers** between **1 and 50**.\
5.  Print the **multiplication table of 5** (5 × 1 to 5 × 10).

------------------------------------------------------------------------

## 📗 Section 2: Loop with Conditions

6.  Print numbers from **1 to 100**, but skip numbers divisible by
    **3**.\
7.  Print numbers from **1 to 50**; if a number is divisible by **5**,
    print **"Buzz"** instead.\
8.  Count how many numbers between **1 and 100** are divisible by
    **7**.\
9.  Print all numbers between **1 and 100** that are divisible by **both
    3 and 5**.\
10. Print the **sum of numbers from 1 to 100**.

------------------------------------------------------------------------

## 📙 Section 3: Number Logic Problems

11. Find the **sum of digits** of a given number\

-   Example: `123 → 6`\

12. Reverse a given number\

-   Example: `123 → 321`\

13. Check whether a given number is a **palindrome**.\
14. Count the **number of digits** in a given number.\
15. Find the **largest digit** in a given number.

------------------------------------------------------------------------

## 📓 Section 5: Advanced Logical Thinking

21. Check whether a number is a **prime number**.\
22. Print all **prime numbers between 1 and 100**.\
23. Find the **factorial** of a given number.\
24. Print the **first 10 Fibonacci numbers**.\
25. Check whether a number is an **Armstrong number**.

------------------------------------------------------------------------

## ✅ Practice Guidelines

-   Solve each problem using **`for` loop** first.\
-   Then solve the same problem using **`while` loop**.\
-   Do **dry-run** for at least **5 problems**.\
-   Write **logic step-by-step** before coding.
